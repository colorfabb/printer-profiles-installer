# v1.6.20 - Windows Installer

## Added in v1.6.20
- AnyCubicSlicer support (AnycubicSlicerNext): installs filament and process profiles into all detected per-account folders under `%APPDATA%\\AnycubicSlicerNext\\user\\<digits>\\` (falls back to `user\\default` if no numeric folders exist).

## Installation
Download and run `colorFabbInstaller_v1.6.20.exe` on Windows.

See `colorFabbInstaller_v1.6.20.sha256.txt` for verification.
